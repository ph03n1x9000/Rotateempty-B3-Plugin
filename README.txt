Rotateempty-B3-Plugin
---------------------

Checks if any scores were made in 3 minutes, if not it will rotate map. For CoD4 server stalls problem


Installation:

1) Place rotateempty.py in the extplugins folder.
2) Edit your b3 config file. If you use b3.ini, add the following to the plugin section.
          rotateempty: None
if you use b3.xml, add the following to the plugins section:
          <plugin name="rotateempty" />
3) Restart B3


Important:
This will only work with B3 version 1.10.x
